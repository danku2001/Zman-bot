# טקסטים מומלצים ל-BotFather

## Description

ZmanBot הוא בוט תזכורות אישי בעברית. כותבים טבעי כמו "מחר ב-9 תזכיר לי לשלוח מייל" או "כל חודש ב-1 לשלם שכירות", והבוט מזכיר בזמן.

## About

עוזר תזכורות חכם בעברית לטלגרם: תזכורות חד-פעמיות, קבועות, דחייה, חיפוש, ביטול, סיכום יומי ושבועי. עובד לפי Asia/Jerusalem.

## Commands

```text
start - התחלה
help - עזרה ודוגמאות
id - הצגת Chat ID
list - כל התזכורות
today - תזכורות להיום
week - תזכורות לשבוע
recurring - תזכורות קבועות
overdue - תזכורות באיחור
search - חיפוש תזכורות
stats - סטטיסטיקות
```

## דוגמאות קצרות

```text
תזכיר לי עוד שעה להתקשר לאמא
מחר ב-9 תזכיר לי לשלוח חשבונית
כל יום ראשון ב-10 לבדוק דוחות
כל חודש ב-1 לחודש לשלם שכירות
מה יש לי היום?
מה באיחור?
בטל את התזכורת לשלוח מייל
```
